## Installation (2 minutes)

```bash
# 1. Clone the repository
git clone https://github.com/Krishankant-Dixit/On-Chain_Message_Board-PublicNote-.git
cd On-Chain_Message_Board-PublicNote-

# 2. Install dependencies
npm install

# 3. Start the development server
npm start
```

## Testing on Your Phone (1 minute)

1. **Open Expo Go** on your phone
2. **Scan the QR code** displayed in your terminal
3. **Wait for the app to load**

## Using the App (2 minutes)

### Step 1: Connect Wallet
- Tap the **"Connect Wallet"** button
- The app will connect with a demo wallet address

### Step 2: View Messages
- Browse the message feed
- Pull down to refresh messages
- See sender addresses and timestamps

### Step 3: Post a Message
- Tap **"Post Message"** at the bottom
- Enter your message (max 280 characters)
- Tap **"Post Message"** to submit
- Confirmation will appear

## Troubleshooting

**Can't connect to server?**
- Make sure phone and computer are on same WiFi
- Restart the Expo server

**App crashes?**
- Check you installed all dependencies
- Try `npm cache clean --force` and reinstall

**Can't scan QR code?**
- Make sure Expo Go is installed
- Try typing the URL manually in Expo Go
